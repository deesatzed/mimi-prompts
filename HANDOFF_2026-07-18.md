## What This Project Does
MiniPromptLib is a personal, self-improving library for managing "mini-prompts" — reusable
instructional snippets ("act as senior Python reviewer", "focus on async race conditions",
"never log PHI") — so they persist across sessions and across different AI coding tools
(Claude Code, Grok Build, local Ollama agents, CAM-PULSE) instead of being retyped from memory
every time.

**Tech Stack:** Python 3.13 (stdlib only — `json`, `re`, `uuid`, `datetime`, `pathlib`,
`argparse`, `typing`). Optional integrations: `ollama` package (local LLM), any
OpenAI-compatible client (xAI/Grok, Together, etc.) — both are bring-your-own, not installed.
**Architecture Pattern:** Single-class library + thin CLI wrapper. No server, no database —
one JSON file is the entire persistence layer.
